This document describes the HydrOffice QC Tools application (1.4). For the project page, go here.

2.1.1. OverviewÂ¶

The Survey Review tab will:

  • Ingest bathymetric grids and feature files (see Data inputs).
  • Scan grids for anomalous grid data “fliers” (see Detect fliers).
  • Scan grids for empty grid nodes “holidays” (see Detect holidays).
  • Computes basic grid statistics to ensure compliance to uncertainty and density requirements “grid qa” (see Grid QA).
  • Scan features to ensure proper attribution (see Scan features).
  • Ensure surveyed features are properly accounted for in the gridded bathymetry (see VALSOU checks).
  • Export bottom samples to a text file for archival (see SBDARE export).

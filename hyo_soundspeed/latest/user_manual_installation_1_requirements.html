This document describes the HydrOffice Sound Speed library and manager (3.0). For the source code, go here.

2.1.1. RequirementsÂ¶

If you decide to install the package in a Python environment, the dependencies are:

  • gdal
  • matplotlib
  • netcdf4
  • numpy
  • pyproj
  • PySide (only for the application)

If you download the frozen application (download page), you don’t need to care about dependencies.
